Quick Start

  1. Preview your webcam using the "Preview" button, and rotate it if needed using the adjacent dropdown. Press Enter to escape the preview.
  2. Select the color of your barbell marker interactively.
    • Press the "Select Color" button.
    • Drag your mouse over the marker in the popup (make sure to move the marker around and select it under varying light conditions and angles).
    • Press Enter when the marker is tracked satisfactorily.
  3. Select the exercise you want to do.
    • Add exercises that you find missing to the /resources/lifts.json file.
  4. Input the weight for the set in lbs or kgs.
  5. Press "Log Set" and wait for the webcam preview to show before lifting.
  6. After lifting, press the Enter key to complete the set.
  7. The results for the set are shown.
